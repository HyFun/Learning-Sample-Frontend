Flexible Box 模型，通常被称为 flexbox，是一种一维的布局模型。它给 flexbox 的子元素之间提供了强大的空间分布和对齐能力。 在 flex 容器中默认存在两条轴，水平主轴(main axis) 和垂直的交叉轴(cross axis)，这是默认的设置，当然你可以通过修改使垂直方向变为主轴，水平方向变为交叉轴，这个我们后面再说。

flex-grow

number
父宽度 > 元素总宽度 500 > 200 + 100 + 100
box1: flex-grow:2; width: 200px;
box2|box3: width: 100px;
计算公式：width = 自身宽度 + ( 父宽度 - (box1宽度 + box2宽度 + box3宽度) ) * (自身flex-grow / 总flex-grow)
box1 = 200 + (500 - (200 + 100 + 100)) * (2/2) = 300
box2 = 100 + (500 - (200 + 100 + 100)) * (0/2) = 100 // 没有设置flex-grow 默认为0
box3 = 100 + (500 - (200 + 100 + 100)) * (0/2) = 100 // 没有设置flex-grow 默认为0
1
2
3
number
父宽度 < 元素总宽度 500 < (1000 + 100 + 100)
计算公式：自身宽度 + (500 - (1000 + 100 + 100)) * (flex-shrink * 自身宽度) / (flex-shrink * box1 + flex-shrink * box2 + flex-shrink * box2)
box1 = 1000 + -700 * (1 * 1000) / (1 * 1000 + 1 * 100 + 1 * 100) = 416.67 box2 = 100 + -700 * (1 * 100) / (1 * 1000 + 1 * 100 + 1 * 100) = 41.67 box3 = 100 + -700 * (1 * 100) / (1 * 1000 + 1 * 100 + 1 * 100) = 41.67
