# Installation

## Mac OS

Use brew

> brew tap mongodb/brew

> brew install mongodb-community@6.0

Verify version

> mongod --version

![](./static/version.png)

Start service

> brew services start mongodb/brew/mongodb-community

Stop service

> brew services stop mongodb/brew/mongodb-community
